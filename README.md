# Project Virgo 🌌

Project Virgo is my personal portfolio website and long-term learning project. It's where I put into practice what I learn, experiment with new ideas, and gradually build a portfolio that I'll eventually use for internships and future job opportunities.

The project shares its name with **Virgo**, my personal Obsidian vault where I document what I learn, keep development notes, and track my progress. This website is the practical side of that learning process.

## Current Status

🚧 Project Virgo is currently in its early stages.

Right now, the website is built with only HTML and CSS. I'm focusing on creating a solid foundation before adding responsiveness, JavaScript, and more advanced features.

## Built With

- HTML5
- CSS3

## Project Goals

Rather than rushing to finish this portfolio, I'm building it as I continue learning web development. Every update reflects something new I've learned along the way.
